# ez-appbar

Application bar component containing features typically required for such:

- Auto-hide/auto-show appbar on scroll.
- Support for auxiliary app bar when scrolling away from original app bar position.
- Configurable scrolling parent element.
- Styling based on theme (primary, secondary, danger, etc.).
- Size variants (small, medium, large).
- Orientation variants (top, bottom)
- Events for appbar, and auxiliary appbar, show/hide states.
- and more.
